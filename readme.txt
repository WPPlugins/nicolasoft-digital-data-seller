Plug-in that supports sales of digital data using Paypal button

== Description ==

A plug-in that has the function of receiving an IPN from Paypal and sending the download link URL of the product file by e-mail when the customer completes the payment from the Paypal button placed on the product sales page

== Installation ==

1. Upload "ns_digital_data_seller_plugin" folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
